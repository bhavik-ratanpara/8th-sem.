# **App Name**: FridgeRecipe

## Core Features:

- Ingredient Input: Allow users to input ingredients they have available.
- Recipe Generation: Generate recipe suggestions based on the user's ingredients, using a generative AI tool.
- Recipe Display: Display the generated recipes in a clear, easy-to-read format, including ingredients and instructions.

## Style Guidelines:

- Primary color: Fresh green (#8FBC8F) to represent freshness and natural ingredients.
- Background color: Light, desaturated green (#F0F8F0).
- Accent color: Earthy orange (#D2691E), for CTAs and emphasis, providing contrast against the green tones.
- Font pairing: 'Poppins' (sans-serif) for headings and 'PT Sans' (sans-serif) for body text.
- Use simple, outline-style icons for ingredients and recipe steps.
- Clean, minimalist layout with a focus on readability and ease of use.
